Automate Flag Retrieval

Got a flag? Don't use the interface, automate it with code.

Use the url above, send a get request to it with the fields id for the user id and flag for the flag you found. The response will return success: true if the flag is valid.

Example url: https://ctf.gradstudent.me/claim?id=62b103dd1fe62ccca6027ed8&flag=46892420854573

HINT: The requests library use a get request requests.get(url,params=params) pass params as dict with the id and flag set.
